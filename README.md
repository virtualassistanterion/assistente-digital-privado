# Landing Page: Assistente Digital Privado

Este repositório contém a página de vendas para o serviço de implementação do **Assistente Digital Privado (OpenClaw)**, focado em profissionais e empresas que buscam automação com privacidade.

## Como hospedar no GitHub Pages

1. Crie um novo repositório no seu GitHub (ex: `assistente-digital`).
2. Faça o upload do arquivo `index.html` para a raiz do repositório.
3. Vá em **Settings** > **Pages**.
4. Em "Build and deployment", selecione a branch `main` e a pasta `/ (root)`.
5. Clique em **Save**.
6. Em alguns minutos, sua página estará online em `https://seu-usuario.github.io/assistente-digital`.

## Estrutura da Página
- **Hero:** Foco em produtividade e proteção de dados.
- **Diferenciais:** Privacidade Total, Integração e Autonomia.
- **Preços:** Setup fixo de R$ 1.200 (Cloud ou Local).
- **Bio:** Perfil técnico (Engenheiro USP).

## Personalização
Para alterar links de contato ou imagens, basta editar o arquivo `index.html`. O design é totalmente responsivo e utiliza Tailwind CSS via CDN para facilitar a manutenção.
